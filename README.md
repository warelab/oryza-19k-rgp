# oryza-19k-rgp

Content for the **Gramene Oryza 19K-RGP** subsite — a guide to accessing, querying, and analyzing the 19K Rice Genome Project (a global atlas of 19,035 rice genomes).

The Markdown here is rendered by the [`gramene-githubdocs`](https://github.com/warelab/gramene-githubdocs) component in [gramene-sites](https://github.com/warelab/gramene-sites); each top-level directory is a navbar section, and subdirectories become navigation levels. Files are ordered by their `NN-` numeric prefix and titled by their front-matter `title:`.

| Directory | Navbar section | Contents |
| --- | --- | --- |
| `Start here/` | Start here | Site map / overview of the atlas and how to access it |
| `Platforms/` | Platforms | SNP-Seek v3, GrameneOryza, Oryza CLIMtools, Code & Models, Archives & Bulk |
| `Workflows/` | Workflows | Three end-to-end example workflows (A, B, C) |
| `API/` | API | Access matrix, the `oryza19k` cookbook, endpoints, precomputed summaries |
| `Resources/` | Resources | Downloads — notebooks, precomputed tables, documentation |
| `Cite/` | Cite | How to cite, and data & code availability |
